HTML Table Reference

Table Tags

Tag Name Description
<table> Table Element Defines the beginning of an HTML table.
<tr> Table Row Element Represents a row within an HTML table.
<th> Table Header Cell Element Defines a header cell within a table row; typically used to label columns or rows.
<td> Table Data Cell Element Represents a standard data cell within a table row.
<caption> Table Caption Element Provides a title or description for an HTML table.
<thead> Table Head Element Groups the header content in a table; typically used to separate headers from the table body.
<tbody> Table Body Element Groups the main content of the table (excluding headers and footers).
<tfoot> Table Foot Element Groups the footer content in a table; typically used for summary information.
<colgroup> Column Group Element Specifies groups of one or more columns in a table for styling or other purposes.
<col> Column Element Defines attributes for one or more columns in a <colgroup> element.

Common HTML Table Attributes

Attribute Description Example
border Sets the border width of the table. <table border="1">
cellspacing Sets the space between table cells. <table cellspacing="5">
cellpadding Sets the space between the content of table cells and the cell borders. <table cellpadding="10">
width Sets the width of the table. <table width="400">
height Sets the height of the table. <table height="300">
align Specifies the horizontal alignment of the table (left, right, center). <table align="center">
bgcolor Sets the background color of the table. <table bgcolor="#FFFF00">
colspan Specifies the number of columns a cell should span. <td colspan="2">
rowspan Specifies the number of rows a cell should span. <td rowspan="3">
